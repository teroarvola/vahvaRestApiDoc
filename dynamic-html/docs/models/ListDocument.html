ListDocument

  • documentType : String
    VAHVA document type identifier.
  • documentTypeNameFi : String
    Written-out name of the document type in Finnish. It has been noted that these \"publicity facing\" document types should be written out using different document type names than those within VAHVA.
  • documentTypeNameSv : String
    Written-out name of the document type in Swedisah.
  • listId : TechnicalId
  • containsPersonalData : Boolean
    Indicates whether the document in question contains personal data or sensitive personal data (SÄHKE2 2.6.7)
  • ordinal : Integer
    The ordinal number of the document in the case
  • documentId : TechnicalId
  • documentFormatType : String
    * 1 - sähköinen versio / electronic document * 2 - ainoastaan paperilla / paper only
    Enum:
    1
    2
  • documentTitle : String
    The name of the document
  • fileName : String
    Name of the file, id + pdf
  • lang : String
    2-letter ISO language code TODO Should be unified across all interfaces, put common types into imported file
    Enum:
    fi
    sv
    fr
    en
    de
  • correctionUnderway : Boolean
    Indicates whether the a correction to the document is underway (but the list item decision has still been written / \"asia on päätöskirjattu\").
  • caseId : TechnicalId
