ListItem

  • caseId : TechnicalId
  • caseTypeName : String
    Case type written out
  • parliamentId : String
    Parliament id, if any
  • presenter : PersonInfo
  • proposal : String
    Presenter's propsal for approval
  • sessionId : TechnicalId
  • additionalInfo : String
    Any additional info for the list item
  • listItemId : TechnicalId
  • minister : MinisterInfo
  • ministryName : String
    The full name of the presenters' ministry
  • ministryShortName : String
    The abbreviation of the presenters' ministry
  • titleFi : String
    The title of the case in the list.
  • additionalDecisionInfo : String
    Additional info regarding the decision taken. After the case has been handled (status = written or above) contains the name of the decision, too.
  • decisionName : String
    The written out decision code
  • titleSv : String
    Title of the list item in Swedish
  • contents : String
    The contents of the list item
  • listStatus : ListStatus
  • jurisdiction : String
    Jurisdiction related to the list item
  • effects : String
    Effects of the case
  • publicEmbargo : String
    Indicates the time when the list item can be made public.
  • documents : List
